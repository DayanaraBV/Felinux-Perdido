# 🐱 Felinux Perdido

Juego interactivo de preguntas de opción múltiple desarrollado para la novatada de la **Facultad de Ingeniería en Electricidad y Computación (FIEC)** de la ESPOL.

---

## 🎯 Descripción

**Felinux-Perdido** es un juego educativo e interactivo donde los usuarios responden preguntas relacionadas con la FIEC, incluyendo su historia, carreras y cultura general.

El objetivo es que los estudiantes, especialmente los de nuevo ingreso, aprendan de forma dinámica mientras ayudan al personaje Felinux a avanzar en el juego.

---

## 🎮 Características

* Preguntas de opción múltiple
* Interfaz interactiva y dinámica
* Pantallas de inicio, juego y resultados
* Sistema de progreso (barra de energía)
* Uso de componentes reutilizables en React

---

## ⚙️ Instalación y uso

1. Clonar el repositorio:

```bash
git clone https://github.com/DayanaraBV/Felinux-Perdido.git
```

2. Entrar al proyecto:

```bash
cd Felinux-Perdido
```

3. Instalar dependencias:

```bash
npm install
```

4. Ejecutar el proyecto:

```bash
npm run dev
```
